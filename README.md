# STACKY 
> An api written in nodeJS, a clone of stackover flow

## Endpoints
`/questions`
> method(s): GET, POST 

`/questions/:qID`
> method(s): GET

`/questions/:qID/answers`
> method(s): POST

`/questions/:qID/answers/:aID`
> method(s): PUT, DELETE

`/questions/:qID/answers/:aID/vote-:dir`
> method(s): POST
